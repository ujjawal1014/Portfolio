## 🖥️ Run Locally

1. **Clone the repo:**
   ```bash
   git clone https://github.com/ujjawal1014/Portfolio.git
   cd Portfolio
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
4. **Open in browser:**
   Visit [http://localhost:5173](http://localhost:5173)
